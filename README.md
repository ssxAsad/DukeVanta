DukeVanta - A fully local, GPU-accelerated LLM environment and Discord Bot Manager. 
I built DukeVanta because I wanted a unified hub to run local AI models without the friction.
No hidden API costs, no cloud telemetry reading your prompts, and no convoluted terminal commands just to get a model talking to a Discord server.
This is a standalone Electron app that lets you download GGUF weights from Hugging Face, 
load them directly into your system's VRAM, and seamlessly bridge them to custom Discord bots—all wrapped in a clean, jet-black and teal glassmorphic UI.
